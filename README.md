アイデア：ゲームの更新速度を一定にして調整する方法。処理開始のタイミングの時間から、全ての処理が終了したところまでの時間を計測する。そして、先に決めた任意の時間と経過時間の差をsleepメソッドを使い処理を停止するでゲーム更新速度を一定にする。
考えられる問題：処理が重くなり任意の時間をオーバーして処理を行う可能性が考えられる。このような場合は、sleepメソッドを無視するように処理をかく。この問題を再現する方法として、処理内部にsleepメソッドを追加して処理時間をオーバーするように処理を停止させれば良い。
